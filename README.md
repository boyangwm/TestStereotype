## How to run

Clone the repo

```
git clone https://github.com/boyangwm/TestStereotype.git
cd TestStereotype
```

Run TestStereotype
```
cd /src/edu/wm/core/
javac Run.java
java Run
```

To check unit tests for the project, please visit

```
cd /src/UnitTests/
```
